# cambridge_search_plugin
## 介紹

![Imgur](https://i.imgur.com/Bl02HYd.png)

Cambridge Search 是一個在google chrome上的一個擴充套件。
幫助使用者可以在任意網頁上，只要可以全選就可以丟到Cambridge收尋上。

提供繁體中文收尋與英文收尋。
特殊英文單字繁體中文沒有收錄，會收尋不到。

## 設定
![Imgur](https://i.imgur.com/fU80Mpy.png)

可以點擊右上角擴充套件icon。
就可以設定搜尋時的語言。
